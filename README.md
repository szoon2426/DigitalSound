# 산스장

웹캠 또는 휴대폰 카메라로 치닝디핑 안의 사용자를 감지하고, 턱걸이/딥스 움직임의 깊이와 반복에 맞춰 산 사운드를 레이어링하는 웹 프로토타입입니다.

## 실행

카메라 권한 때문에 `localhost`나 HTTPS 환경에서 실행해야 합니다.

```bash
python -m http.server 5173
```

Python이 없다면 PowerShell에서 아래 명령을 사용합니다.

```powershell
.\server.ps1 5173
```

브라우저에서 `http://localhost:5173`을 열고 `카메라와 사운드 시작`을 누릅니다.

## 동작 방식

- MediaPipe Pose Landmarker를 CDN으로 불러와 어깨, 손목, 엉덩이, 코 위치를 추적합니다.
- 손목이 어깨보다 위에 있으면 턱걸이, 팔꿈치가 접히고 손목이 몸통 근처에 있으면 딥스로 추정합니다.
- 운동 깊이가 커질수록 바람, 계곡 리듬, 저역 펄스가 추가됩니다.
- 반복이 완료되면 차임 사운드가 울립니다.
- 포즈 모델을 불러오지 못하면 프레임 차이 기반의 움직임 감지 모드로 자동 전환됩니다.

## 배포

정적 파일만 있으므로 GitHub Pages, Netlify, Vercel, Cloudflare Pages에 그대로 배포할 수 있습니다.

휴대폰 카메라를 실제 운동 촬영용으로 쓰려면 HTTPS 배포 URL에서 실행하는 것이 가장 안정적입니다.
